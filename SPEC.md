# Calculator App Specification

## Objective
Develop a simple Flask-based calculator application.

## Functional Requirements
- Accept two numeric inputs
- Perform:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Display calculated result
- Handle division by zero

## Non-Functional Requirements
- Simple UI
- Fast response
- Easy to use

## Tech Stack
- Python
- Flask
- HTML
